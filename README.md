# 🚗 LeiloesTDSat

Projeto desenvolvido como atividade prática para aprendizado de versionamento com Git. 
O objetivo é compreender o funcionamento de commits, branches e histórico de alterações.

## 🛠 Tecnologias
- Java
- MySQL
- Git
- GitHub

## 🚀 Como Executar
1. Clonar o repositório
2. Abrir o projeto na IDE
3. Executar a classe principal
